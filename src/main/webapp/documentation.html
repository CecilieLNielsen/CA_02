"Street": String,

"CityInfo": {
"zipcode": number,
"city": String
}
},
"phones": [{"number": String, "description": String}],
"hobbies": [{"name": String, "description": String}]
}

}

Errors

All errors are reported using this format (with the HTTP-status code matching the number)

{ status : statusCode, "msg": "Explains the problem" }

  • (e1) :{ status : 404, "msg": "No content found for this request" }
  • (e2) :{ status : 400, "msg": "Field ‘xxx’ is required" } (for example, no name provided)
